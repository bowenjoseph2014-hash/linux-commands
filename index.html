🖥️ System Information Commands

Neofetch - Display System Information with Style

Neofetch is a command-line system information tool that displays your OS, kernel, uptime, and more with ASCII art.

Installation on Arch Linux: 
sudo pacman -S neofetch
Installation on Linux Mint: 
sudo apt install neofetch
Installation on SteamOS: 
sudo pacman -S neofetch
Usage: 
neofetch

Output includes:

  • OS and kernel version
  • Uptime
  • Packages installed
  • Shell
  • Resolution and display info
  • CPU and GPU information
  • Memory usage
💡 Tip: Customize your neofetch display with neofetch --help to hide or show specific information.

uname - Unix Name

Display system information:

# Show all system information
uname -a

# Show kernel release
uname -r

# Show hardware platform
uname -m

hostnamectl - Control System Hostname

# Display current hostname information
hostnamectl

# Set a new hostname (requires root)
sudo hostnamectl set-hostname "new-hostname"

📦 Package Management

Pacman - Arch Package Manager

Pacman is Arch Linux's native package manager. Also used in SteamOS (HoloOS) and Arch-based systems.

🔄 Update Operations

# Update package database
sudo pacman -Sy

# Update all packages
sudo pacman -Syu

📥 Install Packages

# Install a package
sudo pacman -S package_name

# Install multiple packages
sudo pacman -S pkg1 pkg2 pkg3

🔍 Search Packages

# Search for a package
pacman -Ss keyword

# Search installed packages
pacman -Qs keyword

# Show package info
pacman -Si package_name

🗑️ Remove Packages

# Remove a package
sudo pacman -R package_name

# Remove with dependencies
sudo pacman -Rns package_name

AUR - Arch User Repository with yay

Install packages from AUR using yay helper (Arch and SteamOS):

Installation: 
sudo pacman -S yay
# Search AUR
yay -Ss keyword

# Install from AUR
yay -S aur_package

# Update all packages including AUR
yay -Syu

APT - Debian/Ubuntu Package Manager

APT is used in Linux Mint and other Debian-based systems:

🔄 Update Operations

# Update package list
sudo apt update

# Upgrade all packages
sudo apt upgrade

# Full upgrade (dist-upgrade)
sudo apt full-upgrade

📥 Install Packages

# Install a package
sudo apt install package_name

# Install multiple packages
sudo apt install pkg1 pkg2 pkg3

🔍 Search Packages

# Search for a package
apt search keyword

# Show package info
apt show package_name

# List available versions
apt-cache policy package_name

🗑️ Remove Packages

# Remove a package
sudo apt remove package_name

# Remove with config files
sudo apt purge package_name

# Auto-remove unused dependencies
sudo apt autoremove

📁 File & Directory Operations

Essential File Commands

ls - List Directory Contents

# Basic listing
ls

# Long format with details
ls -l

# Show hidden files
ls -la

# Sort by modification time
ls -lt

# Human-readable sizes
ls -lh

# Recursive listing
ls -R

mkdir - Make Directories

# Create single directory
mkdir dirname

# Create nested directories
mkdir -p /path/to/nested/directories

# Create with specific permissions
mkdir -m 755 dirname

cp - Copy Files

# Copy file
cp source.txt destination.txt

# Copy directory recursively
cp -r source_dir/ destination_dir/

# Copy and preserve attributes
cp -a source_dir/ destination_dir/

mv - Move/Rename Files

# Move file to directory
mv file.txt /path/to/directory/

# Rename file
mv oldname.txt newname.txt

# Move directory
mv old_dir/ new_dir/

rm - Remove Files

# Remove file
rm filename

# Remove directory recursively
rm -r directory/

# Force remove
rm -f filename

# Interactive removal
rm -i filename

find - Search for Files

# Find by name
find . -name "filename"

# Find by type
find . -type f  # Files only

# Find by size
find . -size +100M

# Find and execute command
find . -name "*.tmp" -exec rm {} \;

⚡ System Performance

htop - Enhanced Process Monitor

A more user-friendly alternative to top for monitoring processes.

Installation: 
sudo pacman -S htop  # Arch/SteamOS
sudo apt install htop  # Mint
htop

Memory & Disk Usage

free - Display Memory Usage

# Display memory in human-readable format
free -h

# Display in megabytes
free -m

# Repeat every 2 seconds
free -h -s 2

df - Disk Space Usage

# Show disk usage
df -h

# Show specific filesystem
df -h /home

du - Directory Space Usage

# Show directory size
du -sh /path/to/directory

# Top 10 largest directories
du -sh * | sort -rh | head -10
💡 Pro Tip: Use watch -n 1 'free -h' to continuously monitor memory in real-time.

🌐 Networking

ip - IP Configuration

Modern networking configuration tool:

# Show all interfaces
ip addr show

# Show IP addresses only
ip addr show | grep "inet "

# Show routing table
ip route show

# Set IP address (requires root)
sudo ip addr add 192.168.1.100/24 dev eth0

ping - Test Network Connectivity

# Ping with default count
ping google.com

# Ping specific number of packets
ping -c 5 google.com

# Ping with timeout
ping -w 2000 google.com

curl - Transfer Data with URLs

# Download file
curl https://example.com/file.txt

# Save to specific filename
curl -o filename.txt https://example.com/file.txt

# Follow redirects
curl -L https://example.com

# POST request
curl -X POST -d "data" https://example.com/api

wget - Download Files

Installation: 
sudo pacman -S wget  # Arch/SteamOS
sudo apt install wget  # Mint
# Download file
wget https://example.com/file.zip

# Resume download
wget -c https://example.com/largefile.iso

# Download multiple files
wget https://example.com/file1.zip https://example.com/file2.zip

🛠️ Useful Utilities

grep - Search Text

# Simple search
grep "pattern" filename.txt

# Case-insensitive search
grep -i "PATTERN" filename.txt

# Recursive search
grep -r "pattern" /path/to/directory/

# Show line numbers
grep -n "pattern" filename.txt

# Count matches
grep -c "pattern" filename.txt

sed - Stream Editor

# Replace first occurrence
sed 's/old/new/' filename.txt

# Replace all occurrences
sed 's/old/new/g' filename.txt

# Delete lines matching pattern
sed '/pattern/d' filename.txt

# Edit file in place
sed -i 's/old/new/g' filename.txt

tar - Archive Files

# Create gzip compressed archive
tar -czf archive.tar.gz file1 file2 directory/

# Extract gzip archive
tar -xzf archive.tar.gz

# List contents
tar -tzf archive.tar.gz

# Extract to specific directory
tar -xzf archive.tar.gz -C /path/to/directory/

chmod - Change Permissions

# Add execute permission
chmod +x filename

# Set specific permissions (numeric)
chmod 755 filename    # rwxr-xr-x

# Recursive change
chmod -R 755 directory/

# Common permissions:
# 755 - Owner: rwx, Group: rx, Others: rx
# 644 - Owner: rw, Group: r, Others: r
# 700 - Owner: rwx, Group: none, Others: none

🔧 Arch Linux Specific Tools

reflector - Arch Mirror Manager

Optimize Arch Linux package mirrors for faster downloads:

Installation: 
sudo pacman -S reflector
# Find fastest mirrors
sudo reflector --latest 20 --sort rate --save /etc/pacman.d/mirrorlist

# Find mirrors from specific country
sudo reflector --country US --latest 10 --sort rate --save /etc/pacman.d/mirrorlist

systemctl - Manage System Services

# Start service
sudo systemctl start service_name

# Stop service
sudo systemctl stop service_name

# Restart service
sudo systemctl restart service_name

# Enable service at boot
sudo systemctl enable service_name

# Check service status
systemctl status service_name

# List all services
systemctl list-units --type=service

journalctl - View System Logs

# Show recent logs
journalctl

# Show last 20 lines
journalctl -n 20

# Follow logs in real-time
journalctl -f

# Show logs for specific service
journalctl -u service_name

# Show logs since last boot
journalctl -b

base-devel - Development Tools

Essential development tools for Arch Linux:

Installation: 
sudo pacman -S base-devel

Includes: gcc, make, patch, pkg-config, and more.

🎮 SteamOS Tools & Commands

About SteamOS

SteamOS (HoloOS) is an Arch-based Linux distribution designed for gaming on Steam Deck and desktop. It inherits Arch's pacman package manager while being optimized for gaming.

Proton - Game Compatibility Layer

Proton is a tool that allows Windows games to run on Linux through Steam.

# List available Proton versions
proton --version

# Check Proton installation location
find ~/.steam/root -name "Proton*" -type d

# Run a game with specific Proton version
PROTON_VERSION="experimental" steam steam://rungameid/GAMEID

Flatpak - Application Management

Flatpak is a universal package format available on SteamOS:

Installation: 
sudo pacman -S flatpak
# Install a flatpak application
flatpak install flathub com.example.app

# Run a flatpak application
flatpak run com.example.app

# List installed flatpaks
flatpak list

# Update all flatpaks
flatpak update

Steam Deck Specific Commands

# Check battery status
cat /sys/class/power_supply/BAT1/capacity

# Check Steam Deck model
cat /proc/device-tree/model

# Access Deck UI (if on desktop mode)
steamos-gamemode

# Disable GPU turbo (for lower power consumption)
echo 0 | sudo tee /sys/devices/system/cpu/cpu0/cpufreq/boost

SteamOS Package Management

# Update system (SteamOS)
sudo pacman -Syu

# Install from official repositories
sudo pacman -S package_name

# Search for packages
pacman -Ss keyword

# Create pacman hook directory
mkdir -p ~/.local/share/hooks
💡 Tip: On Steam Deck, press the power button for a few seconds and select "Switch to Desktop" to access a traditional Linux desktop environment.

🌿 Linux Mint Tools & Commands

About Linux Mint

Linux Mint is a Debian/Ubuntu-based distribution known for its user-friendly interface and stability. It uses APT for package management.

Cinnamon Desktop Environment

Mint's default desktop environment:

# Restart Cinnamon (if frozen)
killall cinnamon && cinnamon &

# List installed Cinnamon extensions
cinnamon-extension-tool -l

# Check Cinnamon version
cinnamon --version

Software Manager - GUI Package Management

Linux Mint includes a graphical software manager, but you can also use:

Launch from terminal: 
mintinstall  # Opens Software Manager

Linux Mint Update Manager

# Check for updates
sudo apt update && sudo apt upgrade

# Launch update manager GUI
sudo xdg-open /usr/bin/mintsystem &

# Auto-install critical security updates
sudo apt install unattended-upgrades

Snaps - Alternative Package Format

Linux Mint supports Snap packages (optional install):

# Install snapd
sudo apt install snapd

# Install a snap application
sudo snap install application_name

# List installed snaps
snap list

# Update all snaps
sudo snap refresh

# Remove a snap
sudo snap remove application_name

PPA - Personal Package Archives

Add additional software repositories:

# Add a PPA
sudo add-apt-repository ppa:username/ppa-name

# Remove a PPA
sudo add-apt-repository --remove ppa:username/ppa-name

# List all PPAs
grep -r "^deb.*ppa:" /etc/apt/sources.list*

# Refresh package list after adding PPA
sudo apt update

Timeshift - System Backup & Restore

Linux Mint includes Timeshift for system snapshots:

# Create a system snapshot (GUI)
sudo timeshift-gtk

# Create snapshot via terminal
sudo timeshift --create --comments "System backup"

# List snapshots
sudo timeshift --list

# Restore from snapshot
sudo timeshift --restore

Keyboard Shortcuts in Mint

# Super key opens menu
Super / Windows key

# Open run dialog
Alt + F2

# Show/hide desktop
Super + D

# Lock screen
Super + L

# Switch between workspaces
Ctrl + Alt + Arrow Keys
💡 Tip: Linux Mint is beginner-friendly! Check Settings → System Settings for easy access to display, keyboard, and theme customization.

💡 Pro Tips & Tricks (All Distros)

Create Useful Aliases

Add to your ~/.bashrc or ~/.zshrc:

# List with size
alias ll='ls -lh'

# System info
alias sysinfo='neofetch'

# Update system (works on both Arch and Mint)
alias archupdate='sudo pacman -Syu && yay -Syu'
alias mintupdate='sudo apt update && sudo apt upgrade'

# Find large files
alias largef='find . -type f -size +100M'

# Check disk usage
alias diskusage='du -sh * | sort -rh'

Quick Package Search & Install

# Find and install in one command (Arch)
sudo pacman -S $(pacman -Ssq keyword | fzf --multi)

# Find and install in one command (Mint)
sudo apt install $(apt search keyword | grep -oP '^\K[^/]+' | fzf --multi)

Real-time System Monitoring

# Watch disk usage
watch -n 1 'df -h'

# Watch memory usage
watch -n 1 'free -h'

# Watch network traffic
watch -n 1 'cat /proc/net/dev'
💡 Tip: Create aliases for frequently used commands to save time and reduce typing errors.
